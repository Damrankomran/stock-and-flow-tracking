# stock-and-flow-tracking
* This is a stock and flow tracking app for businesses.
* Back-end: Spring-Boot 2.3.1
* Front-end: Angular 10.0.1 
* Database: MySQL DB

## App

* Manage customers.
* Manage products.
* Manage orders.
